# 主动学习页面间距规范（从截图分析）

## 统计卡片区
- 4个统计卡片横排，高度约 60-70px
- 内部 padding 约 p-3
- 数字 text-2xl 粗体，标签 text-xs

## Tab 栏
- Tab 高度紧凑，text-sm
- Tab 下方直接是内容区，无多余间距

- 标签 Badge text-xs

- 右侧进度条 + 评分区域紧凑

- 各区块间 space-y-3
